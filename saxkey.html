SaxKey

SaxKey is a MIDI wind controller built with a Teensy board and a couple of sensors. It converts breath pressure into a MIDI control signal which can be used to control parameters in software musical instruments (such as volume, vibrato, etc). This allows the musician to perform a software instrument more like a wind player, controlling the notes with the keyboard and the volume with breath, creating a more expressive performance. This is inspired by the EWI (Electronic Wind Instrument) by Akai.
